# 🏦 Bank Management System Using Python

A console-based Bank Management System developed using Python and Object-Oriented Programming (OOP) concepts. This project allows users to create bank accounts, authenticate customers, deposit money, withdraw money, and check account balances through a menu-driven interface.

---

## 📌 Features

- Create a new bank account
- Generate a unique account number
- Customer authentication
- Deposit money
- Withdraw money
- Display account balance
- Menu-driven interface
- Object-Oriented Programming implementation

---

## 🛠 Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- Abstract Classes (ABC)
- Inheritance
- Dictionaries
- Random Module

---

## 🎯 OOP Concepts Demonstrated

### Abstraction
Implemented using Python's `abc` module to define a common banking interface.

### Inheritance
`SavingAccount` inherits from the abstract `Customer` class.

### Encapsulation
Account information is managed within class methods.

### Polymorphism
Abstract methods are implemented differently in child classes.
